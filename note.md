# ロンテン（note読者向け説明）

## このプロジェクトについて

「テフロン加工のフライパンは危険なの？」「着色料は本当に体に悪いの？」

こうした疑問について、ネット上には「絶対危険！」「まったく安全！」という両極端な情報があふれています。しかし、「何が確認されていて、何がまだわからないのか」を根拠とともに整理した情報は少ないのが現状です。

このnoteは、そうした話題についての**判断材料を提供すること**を目的としています。

## 何を目指しているのか

### やること
1. **一次情報を優先する**: 公的機関の評価書・査読論文・系統的レビューを根拠の中核に置き、メーカー情報や個人ブログは補助にとどめる
2. **証拠の強さを明示する**: 「研究がある」「公的機関が整理している」「まだ不確実」を区別して書く
3. **両論を並べる**: 懸念を支持する根拠と、懐疑的・否定的な根拠の両方を省略せずに示す
4. **出典を必ずつける**: すべての主張に参考文献リンクを添える
5. **推論を見せる**: 「なぜその結論になったのか」の過程を、読者が追えるように書く

### やらないこと
- 特定の商品や企業を批判する
- 「これを買えば安心」と誘導する
- 「〇〇すべき」という結論を押しつける

## 品質担保のための取り組み

### 1. 信頼できる情報源を優先して使う
- 各国の公的機関（厚生労働省、WHO、FDA など）
- 査読付きの研究論文
- 系統的レビュー（複数の研究をまとめた論文）

ブログやSNSの個人的な意見、特定企業のPR記事は補助資料として扱い、結論の根拠には置きません。

### 2. AIと人間の二重チェック
このプロジェクトでは、AIを情報収集・整理・論理チェックの補助として活用しています。AIの出力はそのまま採用せず、必ず人間が元の資料と照合し、最終的な内容に責任を負います。

**AIの役割:**
- 膨大な資料から情報を探し出して整理する
- 専門用語をわかりやすい言葉に翻訳する
- 論理の矛盾がないかチェックする

**人間の役割:**
- AIが見つけた情報が正しいか、元の資料と照合する
- 複数のAIの出力を比較して、偏りがないか確認する
- 最終的な責任を負う

### 3. 有志によるレビュー
記事の元データはGitHubで公開しており、誰でも間違いを指摘したり、情報を追加したりできます。Pull RequestやIssueでの修正提案を受け付けています。

## GitHubとnoteの役割

このプロジェクトには、2つの公開場所があります。

| | GitHub | note |
|---|---|---|
| **誰向け？** | 貢献したい人、技術的な背景を知りたい人 | 記事を読みたい一般の方 |
| **内容** | 元データ（Markdown形式）、参考文献リスト、執筆方針 | わかりやすく整形された記事 |
| **参加方法** | Pull Request、Issue で直接修正提案 | コメント欄でフィードバック |

## バージョンについて

各記事にはバージョン番号と最終更新日を記載しています。内容の変更規模に応じて次のように管理します。

- **v1.0.0**: noteで一般公開された最初のバージョン
- **v1.1.0**: 新しい情報や論点を追加したバージョン
- **v1.0.1**: 誤字修正や表現の改善をしたバージョン
- **v2.0.0**: 大きな研究結果が出て、結論が変わったバージョン

## このプロジェクトの限界

- **すべての論文を網羅できるわけではありません**  
  見落としがある可能性があります。

- **科学的知見は更新されます**  
  今日「不確実」とされていることが、新しい研究によって変わることがあります。公的機関の評価に更新があった場合は記事を改訂します。

- **完全な中立は不可能です**  
  どれほど注意しても、情報の選択や表現に何らかの偏りが混入する可能性があります。指摘があればGitHubのIssueかnoteのコメントでお知らせください。
